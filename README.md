# Secure Helpdesk

A staged portfolio project for building a secure IT help desk dashboard with Next.js, GitHub, and Vercel.

## Current Progress

- Stage 1: Basic app pages
- Stage 2: Navigation and layout

## Pages

- Home
- Dashboard
- Tickets
- New Ticket
- Admin
- Security Report

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.
